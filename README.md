# 🌻 Симулятор Фермы Цветов

Игра-симулятор управления цветочной фермой, написанная на Python с использованием Pygame.

## 📖 Описание

В этой игре вы управляете собственной цветочной фермой:
- Выращивайте различные сорта цветов
- Собирайте урожай и получайте доход
- Покупайте новые сорта растений
- Отслеживайте статистику дохода с помощью графиков

## 🎮 Особенности

- **4x4 игровое поле** для выращивания цветов
- **3 сорта цветов** с разными характеристиками:
  - **Аресис** - базовый сорт (бесплатный, время роста: 15 сек)
  - **Элизиум** - средний сорт (100 монет, время роста: 30 сек)
  - **Тарсикс** - премиум сорт (250 монет, время роста: 45 сек)
- Система сохранения/загрузки игры
- Визуализация истории дохода
- Простое и интуитивное управление

## 🚀 Установка

### Требования
- Python 3.x
- pygame 2.6.1
- matplotlib 3.10.8

### Шаги установки

1. Клонируйте репозиторий или скачайте файлы проекта
2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Запустите игру:
```bash
python main.py
```

## 🎯 Управление

- **Левая кнопка мыши** - посадка/сбор цветов
- **Кнопки интерфейса** - покупка сортов, сохранение/загрузка, просмотр статистики

## 💡 Как играть

1. Начните с доступного сорта "Аресис"
2. Сажайте цветы на свободные участки поля
3. Дождитесь окончания времени роста
4. Соберите урожай и получите доход
5. Накопите монеты для покупки новых сортов
6. Расширяйте ассортимент и увеличивайте прибыль!

## 🔧 Технические детали

- Игра использует Pygame для рендеринга графики
- Сохранения хранятся в формате JSON
- Графики генерируются с помощью matplotlib
